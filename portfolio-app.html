Shoes-On

This application was made when I was going to take part in the IT Days competition which was held on campus. This application has almost the same function as Go-Jek or Grab, it's just that this application is devoted to shoe services such as washing, repainting, repairing and shoe soles. This application uses the Java programming language and is created using the Android Studio tool. This application still has many shortcomings, such as not automatically connected between customers and partners.
